## README
掲示板アプリを作りながら学ぶRuby_on_Rails入門

## 環境構築
- rubyのインストール
- sqliteのインストール
- rubyのbinファイルにsqliteを解凍して入れておく

## コマンド確認
- ruby -v
- sqlite --version
- gem -v
- gem install bunndler
- gem install rails
- rails -v
- rails new XXXXX :新しいプロジェクト作成→フォルダごと作成される
- rails server サーバ起動

## コマンド2
- rails generate controller XXXX ：手打ちでも可能だがコマンドでコントローラーを継承して作成できる
